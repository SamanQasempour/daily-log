A lightweight project for automatically recording daily activity logs and pushing them to GitHub.

---

## ✨ Features

- 📅 Daily activity logging
- 🚀 Automatic Git commits
- 📤 Automatic Git push
- 🖱️ Manual upload support
- ⚡ Lightweight
- 💻 Windows compatible
- 🔒 Uses GitHub Personal Access Token
- 📜 Stores activity history in a simple text file

---

---

## 🚀 How It Works

Every time the script runs, it:

1. Records the current date and time.
2. Updates the history log.
3. Creates a Git commit.
4. Pushes the changes to GitHub automatically.

Example:

```text
Boot logged at: Tue Jul 14 08:32:16 +0330 2026
```
